************
Installation
************

Installation is done via /install.php

System Requirements
=======================
* PostgreSQL >= 16
* PHP >= 8.1
* 2 GB RAM
* 5 GB Disk


Running the Installer
=======================

Download GeoLite, unzip, and FTP to your directory of your choice.

Below, we'll install in directory 'geolite'

Navigate to https://yourdomain.com/geolite/install.php and enter the required information:


.. image:: _static/GeoLite-Install-Screen.png

Click "Install GeoLite" to complete the installation


Sample Data
===================

The default installation includes sample Map, Dashboard, Document, and HTML page.
